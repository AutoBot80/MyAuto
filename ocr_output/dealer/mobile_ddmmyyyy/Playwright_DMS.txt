Playwright_DMS (template folder)

Per-customer execution logs are not stored here. Each real Siebel Fill DMS run
overwrites:

  ocr_output/<dealer_id>/<mobile_subfolder>/Playwright_DMS.txt

with UTC timestamped STEP / NOTE / MILESTONE lines, the field values used for
that run, DECISION lines (e.g. customer_found, in_transit), and an [END] line
with the final error (if any).
