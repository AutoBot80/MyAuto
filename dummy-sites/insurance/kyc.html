KYC Verification

After entering mobile, click Verify mobile. If KYC is already on file for this number, proceed without documents. Otherwise upload Aadhaar front, Aadhaar back, and use the same Aadhaar front file again for Customer Photo, then proceed.

Demo: 9694585832 simulates KYC found; any other valid 10-digit number requires document upload.
